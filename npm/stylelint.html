@wavesdean/stylelint-config

支持配套的 stylelint 可共享配置。

安装

需要先行安装 stylelint 和 stylelint-scss：

npm install @wavesdean/stylelint-config stylelint stylelint-scss --save-dev

使用

在 .stylelintrc 中继承本包:

{
  "extends": "@wavesdean/stylelint-config"
}
